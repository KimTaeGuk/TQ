#[Model 1_Login && Board PortPolio]

DB : mysql v5.7.17
Tool : Eclipse Neon.2
WAS : Tomcat v9.0

사용 언어 : JSP, Javascript, HTML, CSS, Ajax, JQuery

[기능 구현]

*********************************회원가입 & 로그인*********************************

1. 로그인 아이디 저장(쿠키 사용)
2. 판매자와 소비자의 회원가입 분리
3. 정규식을 이용한 아이디 및 패스워드 문자 제한
4. jQuery & 부트스트랩을 이용한 로그인 화면 구현
5. 아이디 및 비밀번호 찾기 기능 / 탈퇴 기능 / 수정 기능
6. 회원가입 시 이미지 업로드를 통한 프로필 사진 구현(cos.jar 사용)
   [회원 정보 수정시 이미지 교체 가능]


***************************************게시판***************************************

1. 판매자가 공지사항 글쓰기를 사용하였을 때 상단 글 노출
2. Ajax 를 이용한 댓글 및 대댓글 수정 및 View페이지 출력
3. 자신의 게시판 글에 댓글이 작성되었을 시, 알림 기능
4. 통합 검색 기능
5. 페이징 처리 기능
6. 댓글 좋아요 및 싫어요 기능 구현
7. 게시판 리스트 페이지에서 댓글 갯수 및 조회수 파악 가능
8. 금일에 작성된 글에는 new.jpg가 제목 옆에 출력
9. 글 작성의 시간이 경과됨에 따라 날짜 글 다르게 출력(ex. 1달전 2달전 1년전 2년전)
10. 웹에디터 사용
